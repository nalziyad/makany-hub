# مكاني Hub 🌿

قاعدة معرفة شخصية مبنية على منهج **مكاني — فقه النفس**

## التشغيل المحلي

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

## الرفع على Vercel

1. ارفع المشروع على GitHub
2. افتح [vercel.com](https://vercel.com)
3. اضغط **New Project** → اختر الـ repo
4. Vercel يكتشف Vite تلقائياً → اضغط **Deploy**
